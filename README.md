## Prerequisites
<hr/>

First check for docker and docker-compose , if not present then install it
* Docker & Docker-Compose
  ```sh
  sudo apt install docker.io docker-compose
  ```

  * Docker Version
  ```sh
  docker -v
  ```

  * Docker-Compose version
  ```sh
  docker-compose -v
  ```
